Sobre mim

Meu nome é Guilherme Linhares

Sou um Engenheiro de Software com foco em desenvolvimento backend utilizando Java e o ecossistema Spring. Atualmente, trabalho como Software Engineer na Lamas Destilaria e estou cursando Sistemas de Informação na Pontifícia Universidade Católica de Minas Gerais.

Minha trajetória inclui experiência na criação de APIs RESTful e na aplicação de arquiteturas de microsserviços, sempre buscando as melhores práticas de desenvolvimento como Clean Code, SOLID e TDD.

Estou constantemente aprendendo e explorando novas tecnologias, com interesse particular em sistemas distribuídos, mensageria (Kafka, RabbitMQ), containerização (Docker) e orquestração (Kubernetes), além de familiaridade com serviços AWS. Busco oportunidades para aplicar e expandir meus conhecimentos na construção de soluções de software inovadoras e eficientes.

Projetos em Destaque

Ecossistema de Microsserviços To-Do & Notificações

Este projeto (o mesmo que você está visualizando no GitHub) demonstra um sistema de gerenciamento de tarefas construído com Java, Spring Boot, Docker e Kubernetes (local). Inclui um `todo-service` para o CRUD de tarefas e um `notification-service` que consome eventos de forma assíncrona via RabbitMQ.

Tecnologias e Conceitos Demonstrados:

  • Java 21, Spring Boot 3.x (Web, Data JPA, AMQP)
  • PostgreSQL, RabbitMQ
  • Docker, Docker Compose
  • Kubernetes (Deployments, Services, ConfigMaps, Secrets)
  • APIs RESTful, DTOs, Validação, OpenAPI
  • Comunicação Assíncrona
