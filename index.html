Projeto em Destaque

Sistema Inteligente de Gerenciamento de Vagas e Precificação Dinâmica

Um backend para gerenciar estacionamentos de forma autônoma, processando dados de sensores em tempo real (simulando um ambiente IoT) para otimizar a ocupação e o faturamento. O sistema reage dinamicamente a eventos de entrada, estacionamento e saída de veículos, aplicando precificação adaptativa e controle de capacidade dos setores.

Destaques Técnicos e Capacidades Demonstradas:

  • Processamento de fluxos de eventos assíncronos (simulando dados de sensores IoT via webhooks).
  • Java 21, Spring Boot 3.x (Web, Data JPA, Validation, Actuator)
  • PostgreSQL (Dockerizado)
  • Docker, Docker Compose (orquestração do app, DB e simulador)
  • Arquitetura Limpa (aplicação dos princípios para separação de camadas)
  • APIs RESTful (design, versionamento, DTOs)
  • Springdoc OpenAPI (documentação interativa)
  • Tratamento Global de Exceções (@RestControllerAdvice)
  • Integração com Simulador Externo (consumo de API e recepção de Webhooks)
  • Testes Unitários (JUnit 5 & Mockito) para lógica de serviço.
